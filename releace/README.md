# とことこ井波マップ - リリースページ

とことこ井波マップのアプリリリースページです。

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## ページ構成

- `/` - ホーム
- `/news` - ニュース
- `/about` - アプリについて
- `/download` - ダウンロード
- `/donation` - 寄付・支援
- `/contact` - お問い合わせ

## 技術スタック

- React 18
- TypeScript
- Vite
- MUI (Material-UI)
- React Router
